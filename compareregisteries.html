Why Global Registries Matter

Making research data repositories visible and discoverable is fundamental to the FAIR principles. Global registries like re3data.org and FAIRsharing.org serve as essential directories, helping researchers, institutions, and automated tools find and identify trusted data sources. This interactive tool explores the distinct roles and shared features of these two key platforms to help the FAIRagro community strategically enhance the visibility of its repositories.

Key Takeaways for FAIRagro

When to use re3data.org

Register your repository here to gain broad visibility as a trusted data source. It's the primary, high-level directory for repositories themselves. Getting listed in re3data is a critical first step for discoverability.

When to use FAIRsharing.org

Use FAIRsharing to describe the specific data standards, formats, and policies your repository recommends or uses. This provides deeper, contextual information that promotes interoperability and reuse.

A Complementary Strategy

For maximum impact, FAIRagro repositories should be present in **both** registries. Use re3data for repository-level visibility and FAIRsharing to detail the standards and policies that make the data within your repository truly FAIR. Together, they create a comprehensive and machine-readable profile of your data ecosystem.
